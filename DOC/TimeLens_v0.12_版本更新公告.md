# TimeLens v0.12.0 版本更新公告

**发布日期**：2026-04-23
**覆盖迭代**：第十一期（智能提醒与每日摘要）+ 第十二期（页面体验优化）

---

## 第十一期：智能提醒与每日摘要

### 新增功能

- **久坐提醒**：连续工作超过阈值（默认 45 分钟）后推送系统通知，提醒休息
- **碎片化预警**：滑动窗口内应用切换超过阈值时推送注意力分散提醒
- **深度工作标记**：同一应用持续使用超过阈值时推送深度工作提示
- **每日摘要推送**：每天指定时间（默认 18:00）自动推送当日活跃摘要通知
- **设置面板**：新增「智能提醒与每日摘要」配置区域，所有阈值/开关/时间均可自定义

### 技术细节

- 新增 `core::nudge` 模块，10 秒轮询后台线程
- 数据库迁移 v7：新增 `nudge_log` 表（保留历史兼容结构）
- 原生系统通知：macOS 通过 osascript，Windows 通过 PowerShell Toast
- 4 个新 Tauri 命令：智能提醒与每日摘要设置

## 第十二期：页面体验优化

### 今日透视

- 「事实报告摘录」和「数据管线」改为可折叠区域，减少首页占位
- 打断来源柱状图下方直接显示应用名称和次数，无须悬停

### 时间线

- 默认按时间倒序排列，最新会话在最上方
- 新增「分组视图」：相邻同应用会话自动合并为时段块，显示累计时长和会话数
- 支持「分组视图」和「完整视图」一键切换

### AI 助手

- 语言设置同步到后端，AI 助手回答语言跟随界面语言

### 设置页

- 新增「高级设置」区域，采集引擎、OCR 管线参数等开发型选项需开启开发者模式后才显示
- 常用设置（主题、语言、提醒、AI 等）保持直接可见

### 关于页

- 移除「支持与赞助」模块
- 「反馈与建议」新增邮箱联系方式（xuzhefeng222@gmail.com）

### 导航

- OCR 检索从开发工具提升到主菜单
- 移除「复盘」旧入口

## macOS 安装提示

If you see "TimeLens.app is damaged and can't be opened" on macOS, this is usually caused by the system quarantine attribute rather than actual file corruption. Run the command below in Terminal:

```
xattr -rd com.apple.quarantine "/Applications/TimeLens.app"
```
