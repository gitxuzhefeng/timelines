# TimeLens Windows 兼容项目

> **目标**：在保持 **P0 范围**不变的前提下，使 TimeLens 在 **Windows 10 / Windows 11** 上具备与 macOS 同等的采集、截图、聚合、托盘与验证面板能力。  
> **分发**：以 **绿色版（解压即用 ZIP）** 为主；与安装版 **共用同一二进制**，差异在打包与说明文档。  
> **约定**：IPC 字段 `accessibilityGranted` / `screenRecordingGranted` **保留**；Windows 上仅调整**语义映射**与前端**文案**。  
> **版本**：v1.1 · 2026-04-13

## 决策摘要（已确认）

1. **绿色版**：ZIP 分发；数据目录默认仍为 **用户主目录下** `.timelens`（与现有一致），不要求「数据跟 exe」。
2. **托盘**：Windows 必须与 macOS 能力对齐（显示主窗口 / 开始采集 / 停止 / 退出）。
3. **字段名**：不改 `PermissionStatus` 字段名；前端仅改展示文案。
4. **范围**：**不超出 P0**；Windows 不实现浏览器 URL 级采集等 P1 能力。
