# TimeLens Windows · 绿色版与安装包

> **版本**：v2.0 · 2026-04-13  
> **关系**：与 **NSIS 安装包**同源一次构建；差别在分发形式（ZIP vs 安装程序）。

---

## 1. 构建

环境与命令见 [Windows上手.md](./Windows上手.md) 第 5 节。常规：

```bash
cd project
npm ci
npm run tauri build
```

ARM64（WoA）机器建议用 **`project/scripts/tauri-build-windows-msvc.cmd`**。

---

## 2. 产物

- 安装包：多在 `project/src-tauri/target/release/bundle/nsis/`（`*.exe`，具体名称与架构以构建机为准）。  
- **绿色版**：从同一 `bundle` 输出中取**完整应用目录**（含 exe、依赖 dll、资源等），再打成 **ZIP**；不要只打包单个 exe。若当前配置仅生成 NSIS，可从安装包解压或按 Tauri 文档调整 bundle 目标。

---

## 3. 用户使用（绿色 ZIP）

解压到任意目录 → 运行主程序（名称以 `tauri.conf.json` 的 `productName` / `mainBinaryName` 为准）→ 数据仍在 **`%USERPROFILE%\.timelens`**。

---

## 4. WebView2 与 SmartScreen

- **WebView2**：Win11 通常已带；Win10 白屏请装 [Evergreen Runtime](https://developer.microsoft.com/microsoft-edge/webview2/)。  
- **SmartScreen**：未签名 exe 可能被拦截，可按「更多信息 → 仍要运行」；正式分发建议代码签名。

---

## 5. 维护

Tauri / bundler 升级后若输出路径变化，请同步更新 [Windows上手.md](./Windows上手.md) 与本节。
