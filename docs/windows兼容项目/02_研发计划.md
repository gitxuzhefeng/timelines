# TimeLens Windows 兼容 — 研发计划

> **版本**：v1.0 · 2026-03-30  
> **输入**：`01_技术方案.md`、已确认决策（绿色版 / 托盘 / 保留字段名 / P0）

---

## 1. 里程碑概览

| 阶段 | 名称 | 目标 |
|------|------|------|
| W0 | 基线可编译 | Windows 上 `cargo build` / `tauri build` 通过，应用可启动 |
| W1 | Tracker 可用 | 前台窗口 + idle → raw_events / app_switches 链路在 Windows 跑通 |
| W2 | Capture 可用 | WebP 截图、去重、snapshots 表与事件在 Windows 跑通 |
| W3 | 托盘与权限闭环 | 托盘菜单、权限检测、打开设置、前端文案 |
| W4 | 打包与文档 | 绿色 ZIP 流程、用户手册 Windows 章节、CI |

各阶段可在同一 Sprint 内连续完成，按人力并行度调整。

---

## 2. 任务分解（建议工单粒度）

### 阶段 W0 — 基线可编译

| ID | 任务 | 交付/要点 |
|----|------|-----------|
| W0-1 | `storage/mod.rs` 平台分支 | Unix：`0o700`；Windows：仅 `create_dir_all` |
| W0-2 | `Cargo.toml` Windows 依赖 | `[target.'cfg(windows)'.dependencies]` 初版 |
| W0-3 | `tauri.conf.json` | Windows bundle 相关配置；确认图标；scope 与 `%USERPROFILE%` 验证 |
| W0-4 | 本地/CI 验证 | `windows-latest` 至少 `cargo check -p timelens`（包名以仓库为准） |

### 阶段 W1 — Tracker

| ID | 任务 | 交付/要点 |
|----|------|-----------|
| W1-1 | `acquisition/windows.rs` | `sample_front_window`、`idle_seconds` |
| W1-2 | `acquisition/mod.rs` | `#[cfg(windows)] pub use windows::*` |
| W1-3 | `lib.rs` | Windows 启用 `spawn_tracker_thread`；移除仅 drain 的占位线程 |
| W1-4 | 联调 | 验证面板或日志：轮询、切换应用、AFK 行为 |

### 阶段 W2 — Capture

| ID | 任务 | 交付/要点 |
|----|------|-----------|
| W2-1 | Windows 像素采集 | WGC 优先 + GDI 降级（或既定等价方案） |
| W2-2 | `capture.rs` 集成 | `#[cfg]` 分支接入现有 WebP / phash / Writer |
| W2-3 | `lib.rs` | Windows 启用 `spawn_capture_thread` |
| W2-4 | `screen_ok` | Windows 上刷新策略与 `check_permissions` 一致 |
| W2-5 | 联调 | 切换窗口触发截图、手动截图、去重 |

### 阶段 W3 — 托盘、权限 API、前端

| ID | 任务 | 交付/要点 |
|----|------|-----------|
| W3-1 | 托盘 | Windows 构建与 macOS 相同菜单与事件 |
| W3-2 | `PermissionStatus` 发射与 `check_permissions` | Windows 映射语义落地 |
| W3-3 | `open_accessibility_settings` / `open_screen_recording_settings` | Windows 路由 |
| W3-4 | `App.tsx`（及相关） | 平台文案；字段名不改 |
| W3-5 | 窗口聚焦回调 | Windows 上刷新 `screen_ok`（若需要） |

### 阶段 W4 — 打包、文档、CI

| ID | 任务 | 交付/要点 |
|----|------|-----------|
| W4-1 | 绿色版发布说明 | 维护 [绿色版打包说明.md](./绿色版打包说明.md)；有变更时同步版本与路径 |
| W4-2 | 用户手册 | `TimeLens_用户使用手册.md` 增加 Windows 安装与权限说明 |
| W4-3 | CI | `.github/workflows` 增加 Windows job（可选仅 main/tag） |
| W4-4 | 交付清单 | `TimeLens_交付清单.md` 勾选 Windows 产物 |

- 无新增 P0 范围外需求；Known issue 有文档条目。
