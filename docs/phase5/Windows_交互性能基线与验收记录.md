## 2. 复现脚本（优化前与优化后统一）

### 脚本 A：主路径鼠标移动

1. 打开应用，依次进入：`/lens` → `/timeline` → `/intents` → `/settings`。
2. 每页连续移动鼠标 30 秒（覆盖列表区域与按钮区）。
3. 记录卡顿等级：`无感` / `轻微` / `明显`。

### 脚本 B：点击反馈时延

1. 每页执行 20 次按钮点击与切换操作。
2. 记录点击到可见反馈的主观感受：`无感` / `轻微` / `明显`。

### 脚本 C：高频列表操作

1. 在 `/timeline`、`/sessions` 快速滚动并连续切换条目（50 次）。
2. 记录是否出现明显掉帧或交互迟滞。
