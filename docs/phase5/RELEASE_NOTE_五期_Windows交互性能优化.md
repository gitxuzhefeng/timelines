# Release Note：五期 Windows 交互性能优化

**版本**：v0.3.1-phase5  
**日期**：2026-04-17

---

## 本次更新

### 1) Windows 交互性能优化（重点）

- 新增 Windows 平台性能样式分支（仅 Windows 生效）。
- 降低高成本视觉渲染开销：弱化 blur、持续动画与重阴影路径。
- 优化高频交互区域（主导航、时间线、会话列表、弹层预览）的 hover/click 反馈稳定性。
- 合并高频会话刷新触发，减少短时间重复刷新造成的 UI 抖动。

### 2) 文档与验收

- 新增五期 PRD 与里程碑验收计划。
- 新增 Windows 基线与验收记录模板，统一复现脚本与记录口径。
- 已完成本地构建/测试与 lint 校验；Windows 实机流畅度对照按验收脚本补测。

---

## 不变项（确保认知一致）

- 不改变采集、分析与展示的业务语义。
- 不变更核心数据结构与接口契约。
- macOS 路径保持原有体验策略，仅做回归验证。

---

## 已知限制

- Windows 最终体验仍受设备 GPU 性能、驱动状态、WebView2 版本影响。
- 不同硬件之间应做分层对比，不建议跨设备做绝对结论。
