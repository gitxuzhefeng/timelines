## 四、手动 E2E（建议）

1. 本机安装 [Tesseract](https://github.com/tesseract-ocr/tesseract) 并确保 `tesseract` 在 `PATH`（或通过环境变量 `TIMELENS_TESSERACT` 指定）。中文可设 `TIMELENS_OCR_LANG=chi_sim+eng`（视语言包安装情况）。  
2. 设置 → 确认 OCR 说明 → 开启 OCR → 开始采集 → 触发截图。  
3. 会话页：应出现摘要或明确空因；关键词搜索应命中非黑名单、非 redacted 的索引文本。  
4. 勾选「OCR 摘要可出境」后生成 AI 复盘：请求体 JSON 含 `ocr_session_summaries`（需在 BYOK 与 AI 开启前提下自测网络调用）。

## 五、已知限制

| 项 | 说明 |
| --- | --- |
| 引擎 | 首期默认 **Tesseract CLI**，非 macOS Vision / WinRT（与 `rule/TimeLens_TechArch_Phase2_截图增强.md` 中长期原则并存，见 `prd/二期截图增强_技术方案.md`） |
| 依赖 | 未安装 Tesseract 时 OCR 失败，状态区可见错误提示；**不阻塞**截图主链路 |
| Retention | 截图保留策略清空 `file_path` 时，OCR 可能报文件缺失；与一期 retention 行为一致，未在本次扩展级联清理 FTS |
