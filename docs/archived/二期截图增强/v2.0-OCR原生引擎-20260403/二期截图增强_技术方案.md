## 2. 落地摘要

| 主题 | 方式 |
| --- | --- |
| 存储 | `snapshot_ocr`、`session_ocr_context`、`snapshot_ocr_fts`（FTS5） |
| 引擎 | macOS **Vision**、Windows **Windows.Media.Ocr**、Linux **Tesseract（TSV）**；**`OcrEngine`**；**无** ONNX / 跨引擎自动降级（本期） |
| 入队 | `WriteEvent::Snapshot` 成功后 **非阻塞** `try_send` OCR 任务 |
| 闸门 | `ocr_enabled` + 黑名单 + 脱敏；AI 出境另需 `ocr_allow_export_to_ai` |
| AI | 条件满足时注入 **`ocr_session_summaries`**（脱敏文本，无图） |
