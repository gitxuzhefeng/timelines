## 使用前请知悉（半页核心）

1. **是什么**：TimeLens 在本地被动记录窗口切换、应用名称与时长，并在切换时截图（用于情境回溯）；数据默认**仅本机**存储。  
2. **必须权限**  
   - **macOS**：**辅助功能**（读当前窗口信息）+ **屏幕录制**（截图）。  
   - **Windows**：按安装后系统提示授予（与截图/前台窗口采集相关的权限，以实际版本为准）。  
3. **隐私摘要**：不采集按键、剪贴板、密码、网络流量；截图会包含当时屏幕画面，敏感场景请**暂停采集**或后续使用黑名单（若版本已支持）。完整说明见用户使用手册「隐私说明」。  
4. **已知预期**：内测版可能存在 UI 不完整、偶发卡顿或权限引导不清晰；你的反馈会直接进 P0 修复列表。

**装好后**：打开应用 → 按提示授权 → 托盘/主界面确认采集运行 → 正常使用 3～7 天后再回答下方「三问」（也可随时把卡点发给我们）。
